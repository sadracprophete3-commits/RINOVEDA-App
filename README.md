# RINOVEDA App
C'est une application mobile éducative et innovante créée pour transformer l’apprentissage, renforcer la collaboration entre les institutions et promouvoir l’innovation dans le système éducatif haïtien.
Conçue pour les élèves, les enseignants, les écoles, les chercheurs et tous les membres du réseau éducatif haïtien, l’application offre un accès simple, rapide et centralisé à des ressources pédagogiques, scientifiques et professionnelles.

### User Stories
Must have
- [x] L'utilisateur peut creer un compte et se connecter afin d'acceder a l'application.
- [x] L’utilisateur peut consulter et télécharger des cours, manuels et fiches pédagogiques.
- [x] L’utilisateur peut visionner des vidéos éducatives directement depuis l’application.
- [x] L’utilisateur peut utiliser des outils d’autoformation interactifs.
- [x] L’utilisateur peut participer à des quiz pour évaluer ses connaissances et suivre ses progrès.
- [x] L’interface doit être intuitive et accessible sur mobile, tablette et ordinateur.
- [x] L’utilisateur reçoit des notifications pour les nouvelles ressources disponibles.
- [x] L'utilisateur peut lire une section decrivant l'application.
- [x] L'utilisateur peut se deconnecter de l'application.
- [ ] L'utilisateur peut supprimer son compte.

 Maybe
- [x] L'utilisateur peut modifier son profil (Photo,Username, Email,Password)
- [x] L'utisateur peut voir son profil personnel.

 Would be nice to have
- [ ] Les contenus doivent être classés par catégorie (matière, niveau, type de ressource).
- [ ] Les quiz doivent fournir un retour immédiat et des explications pour chaque question.
